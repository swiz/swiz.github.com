Packageorg.swizframework.storage
Interfacepublic interface IEncryptedLocalStorageBean
Implementors EncryptedLocalStorageBean

The IEncryptedLocalStorageBean should be used as the type of the autowired variable. Define the implementation EncryptedLocalStorageBean in a BeanLoader The IEncryptedLocalStorageBean wrappes the functionality of the EncryptedLocalStorageBean to supply easier access and to avoid repetitive code.
