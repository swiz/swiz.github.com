addAsync()method
public static function addAsync(testCase:Object, token:AsyncToken, callback:Function, delay:Number):void

addAsync can be used from TestCases which want to rest an asynchronous action (service call which returns AsyncToken) Usage from TestCase: function testLogin():void { TestUtil.addAsync( this, userDelegate.loginUser("foo", "bar"), onLogin, 5000 ); }

Parameters

testCase:Object — The TestCase itself from which this is called
 
token:AsyncToken — The AsyncToken returned by the service-/delegate-call
 
callback:Function — function to call in the testcase when operation is complete
 
delay:Number — time to wait until the call is considered a failure

delayResultHelper()method 
public static function delayResultHelper(token:AsyncToken, event:Event, showBusyCursor:Boolean):void

Helper method that we can call after a delay to notify token responders of either a fault or result.

Parameters

token:AsyncToken — The AsyncToken instance to call the responder on.
 
event:Event — Either a ResultEvent or a FaultEvent. The type of the event determines which responder to call on the token.
 
showBusyCursor:Boolean — When true, the busy cursor is removed when this method executes.
