registerWindow()method 
public function registerWindow(window:IEventDispatcher):void

If you are using AIR with native windows you have to explicitly register new windows so Swiz can listen for added/removed views to inject beans and create dynamic mediators. Be sure to registerWindow BEFORE you open it to have things autowired correctly

Parameters

window:IEventDispatcher — Window reference, the param should be typed mx.core.Window but to avoid an AIR dependency we simply type it IEventDispatcher because we are only interested in adding event listeners
